# Day of Dragons DockerImage
This is a early version of the Day of Dragons Server Image.

## Configuration
### Environment Variables
- `SERVER_NAME="DayOfDragon-Server by Alphyron"` - This is the name you can give the Running Server.

### Ports
- `27015/tcp`
- `27016/tcp`
- `27015/udp`
- `27016/udp`
- `7777/udp`
- `7778/udp`
- `4380/udp`
